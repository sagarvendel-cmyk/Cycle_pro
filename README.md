# Cycle_pro

A modern web application built with **Next.js 16**, **Tailwind CSS**, and **Supabase**. This project provides a clean UI with components for authentication, profile management, and various study-related features.

## Features
- Server‑side rendering with Next.js 16
- Authentication flow using Supabase
- Responsive UI built with Tailwind CSS and Radix UI components
- Study management pages and study operations
- Theme switching with `next-themes`
- Component library using Shadcn UI primitives

## Getting Started
```bash
# Install dependencies (using pnpm, npm or yarn)
pnpm install  # or npm install

# Run the development server
pnpm dev    # or npm run dev
```

The app will be available at `http://localhost:3000`.

## Scripts
- `dev` – Start the development server
- `build` – Build the app for production
- `start` – Run the production build
- `lint` – Run ESLint
